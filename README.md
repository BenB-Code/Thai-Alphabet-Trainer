## Concept

**Thai Flashcards** est ne d'une envie simple : apprendre l'alphabet thai sans prise de tete.

Pas de compte a créer, pas de gamification artificielle, pas de système de points. Juste des **cartes**, un **timer**, et la repetition. L'idée, c'est de reproduire l'expérience d'un paquet de flashcards physiques, en plus pratique : on choisit exactement les lettres qu'on veut travailler, on règle la vitesse, et on lance.

L'application couvre les **44 consonnes** et **39 voyelles** de l'alphabet thai, organisées par classe (low, mid, high) et par type (courte, longue). Chaque carte montre le caractère, sa romanisation, et un mot exemple pour ancrer la memorisation.

> *L'objectif : un outil d'apprentissage sobre, rapide, et utilisable partout -- desktop comme mobile.*

## Fonctionnalites

<table>
<tr>
<td width="50%">

### Quiz par flashcards

- Retourne la carte pour reveler la réponse
- Timer automatique configurable (2s a 10s, ou manuel)
- Barre de progression animée en temps reel
- Animations de transition entre les cartes

</td>
<td width="50%">

### Selection granulaire

- Choisis lettre par lettre ou par categorie entiere
- Consonnes par classe : Low / Mid / High
- Voyelles par type : Short / Long
- Compteur de selection en temps reel

</td>
</tr>
<tr>
<td width="50%">

### Modes d'affichage

- **Thai -> Latin** : reconnaitre le caractère thai
- **Latin -> Thai** : retrouver le caractère depuis la romanisation
- **Mixte** : les deux melanges aléatoirement

</td>
<td width="50%">

### Personnalisation

- Theme clair / sombre
- 3 polices thai au choix (Sarabun, Kanit, Sriracha)
- Interface bilingue français / anglais
- Preferences persistées automatiquement

</td>
</tr>
<tr>
<td width="50%">

### PWA

- Installable sur mobile et desktop
- Fonctionne hors-ligne une fois chargée
- Service Worker pour le cache des assets

</td>
<td width="50%">

### Responsive

- Adapte pour desktop, tablette et mobile
- Utilisable en déplacement, dans les transports
- Interface épurée pour se concentrer sur l'essentiel

</td>
</tr>
